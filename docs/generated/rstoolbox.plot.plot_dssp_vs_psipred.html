rstoolbox.plot.plot_dssp_vs_psipred¶

rstoolbox.plot.plot_dssp_vs_psipred(df, seqID, ax)¶

Generates a horizontal heatmap showing differences in psipred predictions to dssp assignments.

The dssp and psipred strings should be present in the DesignSerie. If not the case, consider computing them using for example the get_sequence_and_structure(). Note that this function will only plot the heatmap for a single decoy. If more are requested, please see the example below.

Parameters:
  • df (Series) – Data container, where ONE column cotains the dssp assignments and a second column the psipred predictions.
  • ax (Axes) – matplotlib axis to which we will plot.

return: list() of Axes

See also

positional_sequence_similarity_plot()
