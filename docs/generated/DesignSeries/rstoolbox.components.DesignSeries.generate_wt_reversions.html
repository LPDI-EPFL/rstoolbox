rstoolbox.components.DesignSeries.generate_wt_reversions¶

DesignSeries.generate_wt_reversions(seqID, key_residues=None)¶

Generate all variant that revert decoy sequences to the reference_sequence.

Expand the selected sequences by generating all the combinatorial options to revert to the reference (WT) sequence.

Alters the names of the designs in description column.

Warning

This is a computationaly expensive function. Take this in consideration when trying to run it. When trying to revert too many mutations, it can run out of memory.

Parameters:
  • seqID (str) – Identifier of the sequence of interest.
  • key_residues (Union[int, list() of int, str, Selection]) – Residues of interest
Returns:

DesignFrame
