rstoolbox.components.SequenceFrame¶

class rstoolbox.components.SequenceFrame(*args, **kwargs)¶

Per position frequency occurrence for a set of decoys.

The SequenceFrame extends pandas.DataFrame. Ideally, in this pandas.DataFrame, each column represents a ResidueType (or NucleotideType) and each row a sequence position.

A reference_sequence can be provided, but it has to be the exact same length as the sequence expressed in the pandas.DataFrame.

As it represents sequence, unless the SequenceFrame is generated by slicing another, one should expect the index to start in 1.
